# Booksy: El mayor mercado de libros en linea de la comunidad ucabista

## Objetivo

Booksy es ofrecerle a la comunidad de estudiantes de la UCAB un sitio seguro y confiable donde puedan vender y comprar libros.

Los estudiantes podrán ingresar a Booksy como visitantes y ver la oferta de libros disponibles. Luego podrán registrarse en el sistema para comprar o vender los libros que quieran. También podrán visualizar los libros en tendencia, buscar libros en especifico y calificar los libros comprados. 

Booksy tiene muchas posibilidades y oportunidades para ser (tecnologicamente hablando) el sistema más innovador. Sin embargo, la maxima prioridad es que Booksy sea un software estable y de calidad para el publico.

## Tecnologias 

Booksy es un software web, por lo que, usaremos tecnologias especializadas en el desarrollo web. A continuación, se detallarán las tecnologias usadas:
- **Vue**: Framework orientado a objetos para el desarrollo de frontend de sitios web. Basado en Javascript y Typescript.
- **.NET**: Framework orientado a objetos para el desarrollo de backend de sitios web. Para ser especifico, se usará **ASP.NET Core**. Basado en C#. 
- **Git**: Controlador de versiones de proyectos de software. Ayudará a mejorar el flujo de trabajo del equipo. Tiene soporte de alojamiento con servicios como Github y Gitlabs.
- **JSON**: Formato de archivos legible y sencillo que será usado para poder establecer una comunicación en el modelo *Cliente-Servidor*. También se usará para el almacenamiento de datos de usuario, productos, entre otros (como si fuera una base de datos).
